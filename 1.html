Getting setup

to get started you first need to download an ide of your choice, and while not entirly nessacery it will make the process of programing less headache inducing

list of suggested ide

  • visual studio - great but can be slow at times when compiling
  • vs code - light weight and fast but lacks built in features such as autocomplete that require addons to be instaled
  • VIM - highly configable and built for linux and macos but can be hard for new programers to learn
