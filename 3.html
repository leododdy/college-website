public vs private

public and private determins the access level of varibles and methods within a class

what this means is when you want to use a varible or a method from another class it needs to be public and if you want to hide a varible in another classes and stop it from being edited from outside sources it needs to be set to be private

it is common practice for a varible to be set as private and have a method setup for fetching the data or setting the data when needed
