if statments

an if statment is a part of the code that makes a choice based on criteria you set

an example of a choice is to see what number a person has input or if a password is correct or if a bool is true or false

an if statment is setup by first writing "if" followed by brackets containing what you want to check with 2 = inbetween what you are checking eg if(1 == 1)
