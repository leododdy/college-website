Variables

varibles are representations of bits of data that your program will use when doing actions such as math and password verification

there are 4 main varibles plus some variations of them

  • Int - any whole number eg 1,2,3,4,0,-1,-2,-3,-4
  • Float - any number with a decimal eg 1.5,-1.5,0.222222,0
  • String - a collection of letters that can spell words eg "njsdj" or "egg"
  • Bool - a true or false statment that can only ever be on or off like a light switch
