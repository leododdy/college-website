methods

methods are parts of code that can be called (called means to run that bit of code) at any point during the running of the application

a method is called by writing its name followed by brackets that may include data that you want to pass into the method for it to use

a method is setup by first declaring if its private or public and then setting the return type of the data you want the method to output with void being no output and then you name the method and follow the name with brackets containing any data the method may use
